INAE: Prevención de deserción estudiantil

Como muestra de nuestra capacidad resolutiva, desarrollamos INAE, una plataforma integral diseñada para instituciones de educación superior.

Combina monitoreo emocional, organización académica y mentorías entre pares, permitiendo a las Direcciones de Asuntos Estudiantiles (DAE) detectar alertas de riesgo en tiempo real y actuar de forma preventiva.

  • ✓ Dashboard analítico para administradores
  • ✓ Gestión segura y confidencial de datos
  • ✓ Sistema automatizado de alertas de riesgo
Desafío

Centralizar señales académicas y de bienestar para apoyar una detección preventiva.

Solución

Plataforma integral con monitoreo emocional, organización académica y mentorías.

Alcance del prototipo

Preguntas frecuentes

Lo esencial antes de comenzar un proyecto

Respuestas referenciales para orientar una primera conversación. Los plazos y alcances definitivos se acuerdan según cada necesidad.

¿Cuánto puede demorar un proyecto? +

Depende del alcance y la complejidad. Un prototipo o solución acotada puede desarrollarse entre 6 y 10 semanas, mientras que una plataforma con integraciones o varios módulos suele planificarse por etapas. Antes de comenzar se define una estimación, hitos y entregables.

¿Trabajan con empresas pequeñas y emprendimientos? +

Sí. La propuesta se adapta al tamaño y madurez de cada organización. Para empresas pequeñas se puede comenzar con una versión inicial que resuelva la necesidad principal y permita incorporar nuevas funciones cuando el negocio crezca.

¿Quién conserva el código y la solución desarrollada? +

Las condiciones de propiedad y uso se establecen antes de iniciar el proyecto. Como criterio general, el cliente recibe la solución y la documentación acordada, mientras que las licencias de herramientas externas se respetan según sus propios términos.

¿Ofrecen soporte después de la entrega? +

Sí, puede considerarse un período de acompañamiento para estabilización, correcciones y orientación de uso. La mantención evolutiva, nuevas funcionalidades o soporte continuo se definen como un servicio adicional según las necesidades del cliente.

¿Cómo se define el costo de un proyecto? +

El valor se determina según alcance, complejidad, integraciones, plazos y nivel de acompañamiento. La primera etapa consiste en comprender la necesidad para preparar una propuesta clara, evitando presupuestos genéricos que no reflejen el trabajo real.

¿Qué ocurre en la primera conversación? +

Se revisa el problema, los usuarios involucrados, los procesos actuales y el resultado esperado. Con esa información se evalúa la viabilidad, se identifican prioridades y se proponen los próximos pasos sin asumir un compromiso inmediato.
